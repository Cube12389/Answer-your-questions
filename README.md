# 回答你的疑问！
遇到购物选择困难怎么办，不妨来看看，选择困难症狂喜，解决你的疑问！（当然没有的词库你得自己更新awa）

## 使用方法
输入你想买的东西（匹配bin/中同名文件），程序根据词库提问，你只需回答"Yes" or "No"。

## 更新词库？
### 格式
```txt
词条数量
词条1 参数一 参数二
词条2 参数一 参数二
...
```
其中参数一对应用户输入"No"跳转的词条，参数二对应用户输入"Yes"跳转的词条<br>
当两个参数都为-1时就是结束（这个时候你就要给出答案了awa）

特别的，当参数二为-1时，仅在当前词条停留一秒然后跳转到参数一对应词条

> 其实就是一个有向图，每个节点有两个出度，然后直接进行图上遍历

最后，把它放到bin/就大功告成了！
